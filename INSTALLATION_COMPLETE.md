## 🔧 故障排除

### 常见问题

#### 1. 端口占用
```bash
# 检查端口使用情况
lsof -i :3000

# 杀死占用进程
kill -9 <PID>
```

#### 2. 依赖安装失败
```bash
# 清除 npm 缓存
npm cache clean --force

# 重新安装依赖
rm -rf node_modules package-lock.json
npm install
```

#### 3. 配置文件错误
```bash
# 检查配置文件语法
node -c config.js

# 验证配置内容
node -e "console.log(require('./config.js'))"
```

#### 4. 邮件发送失败
- 检查网络连接
- 验证邮箱凭据
- 确认 API 权限
- 查看错误日志
